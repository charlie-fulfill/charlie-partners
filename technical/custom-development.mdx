Most of Charlie's setup requires zero development. There is one area where partner involvement may be needed: **safety stock in Block mode**.

## Why Block mode matters

Charlie's safety stock has two modes. In **Block mode**, sellable inventory is enforced globally — meaning every tool that reads Shopify's native stock data will be affected. This includes storefronts, POS extensions, back-in-stock apps, and any third-party tool connected to Shopify inventory.

When a client activates Block mode, it's important to audit which tools in their stack read native inventory — and ensure each one handles sellable stock correctly.

## Integrations

Charlie is actively building native integrations for the most common tools. **Depict** is currently in progress.

For other tools, the level of effort depends on the tool itself:

- Some work out of the box
- Some require configuration
- Some require development work on the partner's side

Charlie maintains documentation for the most common integration scenarios. For tools like **Klaviyo** (back-in-stock notifications), implementation guides are available in the product docs.
